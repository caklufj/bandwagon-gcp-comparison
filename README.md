## CN2 GIA 是什么，为什么它就是比普通线路快

普通国际线路从美国到中国，数据包要经过多个 AS 节点，高峰期大家挤在同一条路上，丢包、延迟抖动是常态。

CN2 GIA 不一样。路由节点全程走中国电信的高端骨干网，IP 段认 `59.43.` 开头——这是纯正的中国电信精品网节点。它有两个特征：

1. **去程和回程都走 CN2 GIA**，不像 CN2 GT 只是部分优化
2. **与普通用户网络物理隔离**，不会被高峰期的流量挤占带宽

搬瓦工 CN2 GIA-E 套餐（E 代表 ECOMMERCE）在此基础上还把带宽升到了 2.5Gbps，并且三网——电信、联通（走 AS9929 优质路由）、移动（CMIN2 线路）——各自走对应运营商的优化路由，不只对电信友好。

谷歌云没有这套东西。延迟方面，GCP 美西机房到国内大概在 170-230ms 且抖动明显，搬瓦工 CN2 GIA 机房同样美西落地，延迟稳在 160-180ms，晚高峰几乎不掉。
